  • Important registers:
    • Register 1: ADC Active Sample Sequencer (ADCACTSS): ADC#_ACTSS_R
    • Register 2: ADC Raw Interrupt Status (ADCRIS): ADC#_RIS_R
    • Register 3: ADC Interrupt Mask (ADCIM): ADC#_IM_R
    • Register 4: ADC Interrupt Status and Clear (ADCISC): ADC#_ISC_R
    • Register 6: ADC Event Multiplexer Select (ADCEMUX) : ADC#_EMUX_R
      • Only settings 0x0 and 0xF
    • Register 11: ADC Processor Sample Sequence Initiate (ADCPSSI): ADC#_SSPRI_R
    • Register 15,27,28,35: ADC Sample Sequence Input Multiplexer Select (0-3) (ADCSSMUX0-3): ADCn_SSMUX#_R
    • Register 16,29,30,36: ADC Sample Sequence Control(0-3) (ADCSSCTL0-3): ADCn_SSCTL#_R
    • Register 17-20: ADC Sample Sequence Result FIFO(0-3) (ADCSSFIFO0-3): ADC#_SSFIFO#_R
    • Register 58: ADC Clock Configuration (ADCCC): ADC#_CC_R (Only setting 0x0)
    • Sec 5.5, page 352: Register 68: Analog-to-Digital Converter Run Mode Clock Gating Control (RCGCADC): SYSCTL_RCGCADC_R
