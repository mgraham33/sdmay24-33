Interrupt-based Device Interaction

Using interrupts and "One Shot Mode," complete 'CAMERA_HANDLER()' and 'student_main()' so that each time a pixel of data is ready to be read:

  • The ISR increments the number of pixels that have been read
  • 'student_main()' calls print_pixel_count() exactly once each time the ISR has indicated a new pixel is ready to be read

Declare additional variables you believe you need (hint: you will likely need a variable for counting pixels and one for indicating a pixel is ready to be read)

Note: The 'print_pixel_count()' function takes one argument: an int of the number of pixels that have been read.
