Interrupt Configuration Registers

Give a brief explanation of the purpose of each of the following registers:

  • NVIC_ENx_R
  • GPIO_PORTx_IM_R
  • GPIO_PORTx_RIS_R
  • GPIO_PORTx_MIS_R
  • GPIO_PORTx_ICR_R

NVIC_ENx_R
This register allows control over which devices' interrupts can be sent to the CPU.

GPIO_PORTx_IM_R
This register is used to indicate which GPIO interrupts have been locally enabled.

GPIO_PORTx_RIS_R
This register is used to indicate which GPIO interrupts have been locally detected

GPIO_PORTx_MIS_R
This register is used to indicate which GPIO interrupts have been locally detected, if they have also been locally enabled by the IM register.

GPIO_PORTx_ICR_R
This register clears the flags that indicate a specific GPIO interrupt has occurred (i.e. clears corresponding bits in the MIS and RIS registers)
